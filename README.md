# SumitToastDamor


## Deployment

To deploy this project run




Step 1. Add the JitPack repository to your build file

```bash

 	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```


Step 2. Add the dependency

```bash

 	dependencies {
	        implementation 'com.github.SumitDamor:SumitToastDamor:1.1'
	}
```
